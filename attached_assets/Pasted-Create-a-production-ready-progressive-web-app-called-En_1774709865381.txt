Create a production-ready progressive web app called “Endometriosis Toolkit.”

Tech stack:

- Next.js 14 with App Router  
- TypeScript  
- Tailwind CSS  
- Supabase for database and authentication  
- Recharts for charts  
- Deployable to Vercel  

The app must be mobile-first and behave like a progressive web app (PWA) so users can install it on their phone from the browser.

---

APP PURPOSE

The Endometriosis Toolkit helps users track pain, symptoms, medications, and generate reports for doctors.

Focus on a clean, calming health-app design with soft colors, clear typography, and simple navigation.

---

AUTHENTICATION

Use Supabase authentication.

Features:
- Email signup
- Email login
- Persistent login session
- Each user only sees their own data

---

DATABASE SCHEMA (Supabase)

Create these tables with relations to the authenticated user.

users
- id (uuid)
- email
- created_at

pain_logs
- id (uuid)
- user_id (uuid)
- date (date)
- pain_score (integer 1–10)
- pain_location (text)
- pain_type (text)
- notes (text)
- created_at

symptom_logs
- id (uuid)
- user_id (uuid)
- date (date)
- symptom_type (text)
- severity (integer 1–10)
- triggers (text)
- notes (text)
- created_at

medication_logs
- id (uuid)
- user_id (uuid)
- medication_name (text)
- dose (text)
- time_taken (timestamp)
- relief_level (integer 1–10)
- side_effects (text)
- created_at
